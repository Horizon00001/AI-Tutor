# 试卷讲解Demo - 功能扩展建议计划

## 项目现状分析

当前系统是一个试卷处理与讲解平台，核心功能包括：
1. **PDF解析** (MinerU) - 将试卷PDF转换为结构化JSON
2. **题目修复** (DeepSeek) - 使用AI修复和优化OCR结果
3. **PPT生成** - 将题目生成带动画的PPT讲解
4. **相似题生成** - 基于原题生成难度可调的相似练习题
5. **数据管理** - SQLite数据库存储试卷、题目、相似题

## 建议新增功能

### 1. 知识点标签与知识图谱 ⭐⭐⭐⭐⭐
**价值**: 帮助学生建立知识体系，实现精准学习

**功能设计**:
- 自动识别题目涉及的知识点（使用AI分析）
- 为每道题打上知识点标签
- 构建知识点之间的关联关系（前置/后置/相关）
- 可视化知识图谱展示

**数据库扩展**:
```sql
-- 知识点表
CREATE TABLE knowledge_points (
    kp_id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    subject TEXT,  -- 学科
    parent_id TEXT,  -- 父知识点
    level INTEGER,  -- 层级
    description TEXT
);

-- 题目-知识点关联表
CREATE TABLE question_knowledge (
    question_id TEXT,
    kp_id TEXT,
    weight REAL  -- 关联权重
);
```

### 2. 错题本功能 ⭐⭐⭐⭐⭐
**价值**: 核心学习功能，帮助学生针对性复习

**功能设计**:
- 学生可标记错题
- 自动记录错误次数、错误时间
- 错题分析（错误类型：概念不清/计算错误/审题错误等）
- 错题复习计划（基于艾宾浩斯遗忘曲线）
- 从错题生成专项练习

**数据库扩展**:
```sql
CREATE TABLE wrong_questions (
    id TEXT PRIMARY KEY,
    user_id TEXT,
    question_id TEXT,
    error_type TEXT,  -- 错误类型
    error_note TEXT,  -- 错误笔记
    wrong_count INTEGER DEFAULT 1,
    first_wrong_at TIMESTAMP,
    last_wrong_at TIMESTAMP,
    is_mastered BOOLEAN DEFAULT FALSE
);
```

### 3. 智能组卷功能 ⭐⭐⭐⭐
**价值**: 根据需求自动生成练习卷

**功能设计**:
- 按知识点筛选组卷
- 按难度分布组卷（如简单:中等:困难 = 3:5:2）
- 按题型组卷
- 智能推荐组卷（基于学生薄弱知识点）
- 支持从相似题库中选题

**API设计**:
```python
POST /api/v1/exams/generate
{
    "user_id": "xxx",
    "name": "周末练习卷",
    "knowledge_points": ["kp1", "kp2"],
    "difficulty_distribution": {"easy": 3, "medium": 5, "hard": 2},
    "total_questions": 10,
    "include_similar": true
}
```

## 优先级建议

| 优先级 | 功能 | 原因 |
|--------|------|------|
| P0 | 错题本功能 | 核心学习场景，用户需求最强烈 |
| P0 | 知识点标签 | 为其他功能（组卷、分析）打基础 |
| P1 | 智能组卷 | 与相似题功能互补，形成完整闭环 |
| P1 | 学习数据统计 | 提升产品价值，增强用户粘性 |
| P2 | 题目收藏与笔记 | 提升用户体验 |
| P2 | 多格式导出 | 实用功能，实现相对简单 |
| P3 | 试卷对比分析 | 锦上添花功能 |
| P3 | 学生能力评估 | 需要较多数据积累 |
