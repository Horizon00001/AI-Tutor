# 自动存库功能实现计划

## 需求明确

**只需要存储到数据库的内容：**

1. **DeepSeek输出的题目** → 存入 `questions` 表
2. **相似题目** → 存入 `similar_questions` 表

**不需要存数据库的内容：**

* raw_json 路径（仅文件系统存储）

* ppt_path（仅文件系统存储）

***

## 当前JSON结构

**DeepSeek输出的题目（processed_json）结构：**

```json
{
  "title": "题目编号或大题标题",
  "source": "题目来源/试卷名称",
  "content": "完整题干（LaTeX格式）",
  "answer": "参考答案",
  "analysis": "知识点解析"
}
```

**相似题目结构（similar_questions）：**

```json
{
  "task_id": "...",
  "source_question": {...},
  "similar_questions": [
    {
      "title": "...",
      "content": "...",
      "answer": "...",
      "analysis": "...",
      "difficulty": "same/easier/harder"
    }
  ]
}
```

***

## 数据库结构分析

### similar_questions 表已有字段：
- `similar_id` - 相似题ID
- `source_question_id` - 原题ID（关联 questions 表）
- `exam_id` - 试卷ID
- `user_id` - 用户ID（用户隔离）
- `title`, `content`, `answer`, `analysis`, `difficulty`
- `created_at`

**结论：**
1. 相似题目已经通过 `source_question_id` 关联原题
2. 已经有 `user_id` 字段做用户隔离

**修改 run_pipeline() 函数**:

```python
async def run_pipeline():
    try:
        # MinerU提取...
        result = await mineru_service.extract(file_path, mineru_task.task_id)

        if result:
            pipeline["stages"]["mineru"]["status"] = "completed"

            # DeepSeek修复后 - 创建试卷并提取题目
            fixed_result = await deepseek_service.fix_json(result, deepseek_task.task_id)

            if fixed_result:
                pipeline["stages"]["deepseek"]["status"] = "completed"
                pipeline["stages"]["deepseek"]["result_file"] = str(fixed_result)

                # 初始化数据库并创建试卷记录
                init_db()
                exam_result = create_exam_record(
                    user_id=user_id,
                    source=file.filename,
                    raw_json_path=str(result),
                    cleaned_json_path=str(fixed_result)
                )

                if exam_result["success"]:
                    pipeline["exam_id"] = exam_result["exam_id"]

                    # 提取题目存入 questions 表
                    add_questions_from_json(
                        exam_id=exam_result["exam_id"],
                        user_id=user_id,
                        json_file_path=str(fixed_result)
                    )

                # PPT生成...
            else:
                pipeline["stages"]["deepseek"]["status"] = "failed"
        else:
            pipeline["stages"]["mineru"]["status"] = "failed"

    except Exception as e:
        print(f"Pipeline error: {e}")
```
