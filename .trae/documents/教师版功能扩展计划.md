## 推荐功能（教师视角）

### 1. 知识点标签系统 ⭐⭐⭐⭐⭐
**教师价值**: 精准把握教学重点，快速筛选题目

**功能设计**:
- **自动识别**: AI自动分析题目，提取知识点标签
- **手动编辑**: 教师可调整、补充标签
- **多级分类**: 支持章节→单元→知识点的层级结构
- **标签管理**: 创建自定义标签体系
- **统计功能**: 查看试卷知识点覆盖情况

**教师使用场景**:
- 上传试卷后，系统自动标注每道题考查的知识点
- 教师快速了解试卷结构，把握教学重点
- 按知识点筛选题目，制作专项练习

**数据库设计**:
```sql
-- 知识点表（支持多级）
CREATE TABLE knowledge_points (
    kp_id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    subject TEXT,           -- 学科：数学/物理/化学等
    grade TEXT,             -- 年级
    parent_id TEXT,         -- 父知识点ID
    level INTEGER,          -- 层级：1-章, 2-节, 3-知识点
    description TEXT,
    created_by TEXT         -- 创建者（系统/教师）
);

-- 题目-知识点关联
CREATE TABLE question_knowledge (
    question_id TEXT,
    kp_id TEXT,
    weight REAL DEFAULT 1.0,  -- 关联权重（主要/次要知识点）
    is_auto BOOLEAN DEFAULT TRUE  -- 是否AI自动标注
);

-- 教师自定义标签
CREATE TABLE teacher_tags (
    tag_id TEXT PRIMARY KEY,
    teacher_id TEXT,
    tag_name TEXT,
    color TEXT  -- 标签颜色
);
```

### 2. 智能组卷系统 ⭐⭐⭐⭐⭐
**教师价值**: 快速生成高质量练习卷，减轻备课负担

**功能设计**:

#### 2.1 基础组卷
- **按知识点组卷**: 选择特定知识点，自动生成练习题
- **按难度组卷**: 设置难度分布（基础:中等:提高 = 4:4:2）
- **按题型组卷**: 选择题/填空题/解答题比例设置
- **题量控制**: 设置总题数和分值

#### 2.2 智能推荐组卷
- **薄弱点针对性组卷**: 根据班级错题数据，生成针对性练习
- **相似题扩展**: 选择一道题，自动生成多道相似题组卷
- **变式训练组卷**: 围绕核心知识点，生成不同角度的变式题

#### 2.3 组卷模板
- 保存常用组卷配置为模板
- 支持模板复用和分享
- 内置常见模板（如"周末作业模板"、"单元测试模板"）

#### 2.4 试卷预览与调整
- 实时预览生成的试卷
- 手动替换/删除/添加题目
- 调整题目顺序

**API设计**:
```python
POST /api/v1/exams/generate
{
    "teacher_id": "xxx",
    "exam_name": "第三单元练习题",
    "knowledge_points": ["kp1", "kp2", "kp3"],
    "difficulty_distribution": {
        "easy": 4,      -- 基础题
        "medium": 4,    -- 中等题
        "hard": 2       -- 提高题
    },
    "question_types": {
        "choice": 5,    -- 选择题
        "fill": 3,      -- 填空题
        "answer": 2     -- 解答题
    },
    "source": "all",    -- 题库来源：all/当前试卷/相似题库
    "include_answers": true,   -- 是否包含答案
    "include_analysis": false  -- 是否包含解析
}
```

**数据库设计**:
```sql
-- 组卷模板
CREATE TABLE exam_templates (
    template_id TEXT PRIMARY KEY,
    teacher_id TEXT,
    name TEXT,
    config_json TEXT,  -- 组卷配置JSON
    created_at TIMESTAMP
);

-- 生成的试卷
CREATE TABLE generated_exams (
    exam_id TEXT PRIMARY KEY,
    teacher_id TEXT,
    name TEXT,
    config_json TEXT,
    question_ids TEXT,  -- JSON数组
    created_at TIMESTAMP
);
```

### 3. 题目收藏与管理 ⭐⭐⭐⭐
**教师价值**: 积累个人教学资源库

**功能设计**:

#### 3.1 收藏功能
- **快速收藏**: 一键收藏优质题目
- **收藏分类**: 自定义文件夹（如"经典例题"、"易错题"、"竞赛题"）
- **收藏标签**: 给收藏的题目打标签
- **收藏备注**: 添加教学心得、使用建议

#### 3.2 资源库管理
- **个人题库**: 管理所有收藏的题目
- **搜索筛选**: 按知识点/难度/题型/标签筛选
- **批量操作**: 批量移动、删除、导出
- **资源统计**: 题库总量、各知识点题目数量

#### 3.3 资源共享
- **分享题目**: 生成分享链接给其他教师
- **导入题目**: 从其他教师分享链接导入
- **校本题库**: 学校内部共享题库（可选）

#### 3.4 题目笔记
- 为每道题添加教学笔记
- 记录学生常见错误
- 标注讲解要点

**数据库设计**:
```sql

    name TEXT,
    parent_id TEXT,  -- 支持嵌套文件夹

    created_at TIMESTAMP
);

-- 题目收藏
CREATE TABLE teacher_collections (
    collection_id TEXT PRIMARY KEY,
    teacher_id TEXT,
    question_id TEXT,
    folder_id TEXT,
    tags TEXT,        -- JSON数组
    note TEXT,        -- 教学笔记
    student_common_errors TEXT,  -- 学生常见错误
    teach_points TEXT,  -- 讲解要点
    created_at TIMESTAMP
);
```

### 智能组卷算法
```python
def generate_exam(config):
    # 1. 根据知识点筛选候选题目
    candidates = filter_by_knowledge_points(config.kp_ids)
    
    # 2. 按难度分布选择
    selected = []
    for difficulty, count in config.difficulty_dist.items():
        pool = [q for q in candidates if q.difficulty == difficulty]
        selected.extend(random.sample(pool, min(count, len(pool))))
    
    # 3. 确保知识点覆盖
    selected = ensure_kp_coverage(selected, config.kp_ids)
    
    # 4. 可加入相似题
    if config.include_similar:
        selected = add_similar_questions(selected, config.similar_count)
    
    return selected
```
